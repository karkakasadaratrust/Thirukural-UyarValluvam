## குறள் 
**கற்றதனா லாய பயனென்கொல் வாலறிவ  
னற்றா டொழாஅ ரெனின்**


## குறள் - பிரித்து வாசிக்கும் முறை
**கற்றதனால் ஆய பயனென்கொல் வாலறிவன்  
நற்றாள் தொழாஅர் எனின்**


## தொடர் அமைப்பு
**கற்றதனால் ஆய பயன் என்? வாலறிவன் நற்றாள் தொழாரெனின்**

## உரை
**கற்றதனால் ஆய பயன் என்?** - எல்லா நூல்களையுங் கற்றவர்க்கு அக்கல்வியறிவானாய பயன் யாது?
**வாலறிவன் நற்றாள் தொழாரெனின்** - மெய்யுணர்வினையுடையானது நல்ல தாள்களைத் தொழாராயின்


## உரைக்குறிப்பு
{ **இலக்கணம்** - எவனென்னும் வினாப்பெயர் என்னென்றாய், ஈண்டு இன்மை
குறித்துநின்றது. **கொல்லென்பது** அசைநிலை. }  
பிறவிப்பிணிக்கு மருந்தாகலின் **நற்றாளென்றார்.**  
_ஆகமவறிவிற்குப் பயன் அவன் தாளைத் தொழுது பிறவியறுத்தலென்பது இதனாற் கூறப்பட்டது._


## உரை - பிரித்து வாசிக்கும் முறை
**கற்றதனால் ஆய பயன் என்** - எல்லா நூல்களையும் கற்றவர்க்கு அக்கல்வி அறிவான் ஆய பயன் யாது?
**வால் அறிவன் நல்தாள் தொழாஅர் எனின்** - மெய்யுணர்வினை உடையானது நல்ல தாள்களைத் தொழாராயின்?


## உரைக்குறிப்பு - பிரித்து வாசிக்கும் முறை
{ **இலக்கணம்**  - எவன் என்னும் வினாப்பெயர் என் என்றாய், ஈண்டு இன்மை குறித்து நின்றது.  
**கொல்** என்பது அசைநிலை. }  
பிறவிப் பிணிக்கு மருந்து ஆகலின் **நற்றாள்** என்றார்.  
_ஆகம அறிவிற்குப் பயன் அவன் தாளைத் தொழுது பிறவியறுத்தல் என்பது இதனான் கூறப்பட்டது._
